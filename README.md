Práctica 6: 

Análisis Granulométrico de Suelos

 Problema Resuelto con Asistencia de IA

Desarrollé esta aplicación para automatizar el análisis granulométrico de suelos utilizando inteligencia artificial como herramienta de desarrollo. La IA me ayudó a crear una solución eficiente para un proceso que normalmente es manual y propenso a errores en ingeniería geotécnica.

 Mi Enfoque con IA

Como estudiante, usé IA para:

Generar el código base de la aplicación

Optimizar los algoritmos de cálculo granulométrico

Diseñar la interfaz gráfica profesional

Resolver errores y depurar el código

Documentar el proyecto de manera clara

 Aportes Personales

- Aunque la IA generó el código, yo:

- Comprendí los algoritmos matemáticos

- Ajusté los cálculos según normas de ingeniería

- Integré todas las partes en una aplicación funcional

Funcionalidades Desarrolladas

Con ayuda de IA implementé:
- Procesamiento automático de datos de tamizaje
- Cálculo de porcentajes y clasificación USCS
- Generación de curvas granulométricas
- Sistema de exportación de reportes
- Interfaz visual intuitiva

Lo Que Aprendí

Este proyecto me enseñó a:

- Colaborar con IA para resolver problemas complejos

- Traducir necesidades de ingeniería a código funcional

- Validar resultados computacionales con teoría

- Integrar múltiples tecnologías en una solución

- Presentar soluciones técnicas de manera profesional

Resultado Final

Creé una herramienta que demuestra cómo la IA puede potenciar el trabajo del ingeniero civil, automatizando tareas repetitivas y permitiendo enfocarse en el análisis y toma de decisiones.

La IA no reemplaza al ingeniero, sino que se convierte en una herramienta que amplifica nuestras capacidades para resolver problemas más complejos de manera más eficiente.
