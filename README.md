Fundamentos de Dosificación de Concreto

La dosificación de concreto es el proceso de determinar las proporciones óptimas de cemento, arena, grava y agua para obtener un concreto con las propiedades requeridas de resistencia, trabajabilidad y durabilidad.

Variables Principales:
Resistencia a la Compresión (f'c)

175 kg/cm²: Concreto pobre (cimentaciones, firmes)

210 kg/cm²: Concreto normal (losas, columnas)

250 kg/cm²: Concreto resistente (vigas, trabes)

300 kg/cm²: Alta resistencia (elementos estructurales críticos)

Proporciones Típicas:

1:2.5:3.5 - Para f'c = 175 kg/cm²

1:2:3 - Para f'c = 210 kg/cm²

1:1.5:2.5 - Para f'c = 250 kg/cm²

1:1:2 - Para f'c = 300 kg/cm²

Factor de Volumen Seco (1.54):

Considera la expansión de los materiales secos

1 m³ de concreto fresco ≈ 1.54 m³ de materiales secos
